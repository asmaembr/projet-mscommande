# Microservices Configuration

This project consists of two microservices:
1. **Config Server** - Runs on port `8000`.
2. **CommandeMS** - Runs on port `8001`.

## Port Configuration
| Microservice       | Port  |
|--------------------|-------|
| Config Server      | 8000  |
| Commande Microservice (CommandeMS) | 8001  |
